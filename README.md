# Backend API Server

A simple backend API built with Node.js, Express, and MongoDB (Mongoose).  
Designed to support a small web application with basic data storage and retrieval.

---

## Tech Stack

- Node.js
- Express
- MongoDB (Mongoose)

---

## Setup

Install dependencies:

```bash
npm install
```

---

## Run in Development

```bash
npm run dev
```

The server will start with hot reloading enabled.

---

## Run in Production

```bash
npm start
```

This project does not require a build step.

It runs directly on Node.js.
